## Built entirely on the MERN Stack:
- Fully online database through `MongoDB Atlas` created using the **WordNet API**
- `Express.js` used as communication between the frontend and the Database
- `React.js` (Vite) used to build the frontend
- `Node.js` used to build the backend

## Cloud Deployment Platforms:
- `Render` used to host the backend
- `Vercel` used to deploy the frontend of the page
